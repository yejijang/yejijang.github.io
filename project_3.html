SKHU 졸업 관리 시스템

Web 프로젝트
2018.03 ~ 2018.06 (4달간)

⋆⁺₊⋆주제

졸업 관리 시스템 - 학생들이 졸업 요건을 확인하고, 자신의 교육 이수 현황을 쉽게 파악할 수 있는 시스템

⋆⁺₊⋆개요

SKHU 졸업 관리 시스템은 JSP/Servlet를 이용하여 구현한 웹 프로젝트로, 매년 복잡해지는 졸업요건을 파악하는데 편리하게 사용할 수 있습니다.
학생은 자신의 수업 이력과 성적 파일을 업로드해 그에 따른 교육 이수 현황을 확인할 수 있으며,
여러 졸업 요건과 비교한 결과를 한눈에 보기 쉽도록 구현하였습니다.
교수는 담당 학생의 줄업 요건 이수 현황을 확인할 수 있으며 피드백을 통하여 쉽게 관리할 수 있게 하였고,
관리자 또한 매년 추가되거나 변경되는 졸업 요건을 관리하기 편하게 명확한 UI를 제공하였습니다.

⋆⁺₊⋆담당업무

디자인 총괄, 물리 DB 모델링, Front-End

⋆⁺₊⋆스킬

개발 환경: Windows, JDK1.8, Eclipse, MySQL Workbench, Git
기술 키워드: Java, MySQL, JSP, jQuery, JSTL, HTML5, CSS3, JavaScript, Ajax, MyBatis, Vue.js, JSTL, JSON, Apache POI..

⋆⁺₊⋆최종 후기

이 프로젝트는 JSP, Servlet 을 이용한 웹 프로젝트입니다.
대학교 재학 시절 전면 개편된 복잡한 졸업 요건을 쉽게 파악할 수 있게 하기 위하여 구현하였습니다.
프로젝트 초기부터 학과/학번/특별과정에 따라 다른 졸업 사정을 분석하여 파악하는데 어려움이 있었고,
또한 초기 데이터로 다양한 학번의 수업 이력과 성적 데이터가 필요했기 때문에 데이터 수집하는데도 많은 어려움을 겪었습니다.
당시 팀원 모두 프로젝트를 처음 진행하여 E-R 모델링 도구의 존재를 알지 못해 화이트보드에 테이블과 관계를 정리하며 ER diagram을 그려갔던 기억이 있습니다.

구동 방식은 사용자가 종합 정보 시스템에서 자신의 성적을 Excel 파일로 생성한 후,
졸업 관리 시스템 회원 가입 혹은 개인 정보 수정에 생성한 Excel 파일을 업로드하면
Apache POI를 이용하여 Excel 파일의 내용을 읽어 데이터베이스에 저장하는 방식으로 진행하였습니다.

이 주제는 수강하는 프로젝트 수업에서 동일한 주제로 경쟁하였기에 저희 팀만의 강점으로 4가지를 추가 구현하였습니다.
첫 번째로는 사용자의 편의성을 고려하여 이용 안내 부분을 구현하였습니다.
성적 파일 업로드하는 방법 등과 같이 사진과 설명으로 사용법을 세세하게 나타내어 처음 사용하더라도 쉽게 이용할 수 있게끔 구현하였습니다.
두 번째로는 교사가 학생을 조회하는 부분검색 기능을 구현하였습니다.
하나의 키워드 검색이 아닌 하나 또는 여러 개의 조건으로 사용자가 원하는 상세 검색을 할 수 있도록 편리하게 구현하였습니다.
세 번째는 보다 편리한 UI를 위하여 18학번 이후부터 해당하는 핵심 역량 부분을 시각적으로 나타냈습니다.
단순 텍스트가 아닌 트리구조로 표현하고, 이수 결과를 색깔로 구분해 표시하여 사용자가 보다 편리하게 식별할 수 있도록 구현하였습니다.
마지막으로는 타과 졸업요건을 반영하였습니다.
앞서 말한 것과 같이 졸업요건 자체가 각기 달라 파악하는데 많은 시간이 소요되어, 초기에는 소프트웨어공학과 및 IT 융합 자율 학부 기준으로만 구현하였으나
추가적으로 모든 학우들의 사용을 위하여 타과생들도 사용할 수 있는 시스템을 구현할 수 있었습니다.

이러한 저희 팀만의 강점을 살려 학과 경진대회에서 공통 주제 부문 1등이라는 결과를 얻을 수 있었고,
유지 보수하여 교내 IT 경진대회에서 또한 대상이라는 성적을 거둘 수 있게 되었습니다.
처음 해보는 웹 프로젝트였기에 많이 깨지고 부딪혔으나 새로운 도전과 그 도전을 마쳤을 때 느껴지는 성취감에 개발의 즐거움을 알게 되었습니다.
