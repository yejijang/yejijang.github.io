SKHU_CMS

Spring Web 프로젝트
2020.03 ~ 2020.06 (4달간)

⋆⁺₊⋆주제

성공회대학교 동아리 관리 시스템 SKHU Club Management System - 동아리의 모집에서부터 홍보, 회의록, 회계 등 업무를 관리할 수 있는 시스템

⋆⁺₊⋆개요

SKHU_CMS는 Spring Boot를 이용하여 구현한 웹 프로젝트로, 교내의 모든 동아리 업무 관리에 편리하게 사용할 수 있습니다.
동아리 연합회는 전체 동아리의 정보를 확인하고 관리할 수 있으며,
동아리 관리자는 해당 동아리 관리 및 동아리원 관리를 할 수 있습니다.
또한 동아리원은 소속 동아리의 중요 정보를 확인하고 전체 동아리의 정보를 확인할 수 있습니다.
현재는 배포를 중단한 상태로 소스 코드는 깃허브에서 확인할 수 있습니다.

⋆⁺₊⋆담당업무

DB설계, 동아리 기능 Back-End, Front-End

⋆⁺₊⋆스킬

개발 환경: Windows, JDK1.8, STS, MySQL Workbench, Git
기술 키워드: Spring Boot, MySQL, MyBatis, Maven, AWS, DNS, http, Bootstrap, Gabia, tomcat, AWS EC2, AWS Route53..

⋆⁺₊⋆최종 후기

이 프로젝트는 Spring Boot를 이용하여 구현한 웹 프로젝트입니다.
대학 재학 시절 다수의 동아리와 소모임의 관리를 편리하게 할 수 있는 프로그램의 필요성을 생각하여 구현하게 되었습니다.
기존에는 통일된 방식이 없어 각 동아리의 홍보, 행사 안내 포함 모든 업무를 수기로 작성하거나 SNS를 통해 관리하였는데,
하나의 프로그램으로 모두 관리할 수 있다면 해당 동아리의 관리 및 일반 학우가 여러 동아리의 정보를 한 번에 확인하는 데 있어 편리할 것 같아 진행하게 되었습니다.
모든 동아리를 관리하는 동아리 연합회에 추가적으로 필요한 부분을 문의한 결과,
각 동아리에 지급되는 지원금의 회계를 작성하는 부분이 있으면 관리하는데 편리할 것 같다는 답변을 듣고 추가적으로 구현하게 되었습니다.

Spring Boot, MyBatis를 이용하여 Back-End를 개발하였고, Front에는 Bootstrap 및 템플릿을 이용하여 개발하였습니다.
DBMS로는 MySQL을 이용하여 데이터베이스를 설계하였습니다.
또한, 협업 및 버전 관리를 하기 위해 Git, GtiHub을 이용하여 기능별로 branch를 나누어 작업하였습니다.
branch를 구분하여 작업하는 것은 처음이었기에 헤매긴 하였으나 사용할수록 코드 분리에 편리하다는 걸 알게 되어 중요성을 느끼게 되었습니다.

구현 당시 출석체크 부분을 구현하며 비동기 통신인 Ajax를 처음 접하게 되어 많은 어려움을 겪은 경험이 있습니다.
jQuery 문법도 개념이 제대로 안 잡혀 있던 상태에 모달을 띄우는 것 자체도 힘들었으나
해당하는 동아리의 동아리원 목록을 받아와야 했기에 Ajax를 사용하면서 많은 어려움을 느꼈습니다.
당시 팀원들과 학과 학우들의 도움을 받아 해결은 하였으나, 한동안 모달에 크게 데었다는 생각에 모달 자체를 무서워했던 것 같습니다.

프로젝트 구상 시 실제 구동되는 페이지로 작동되는 걸 최종 목표로 하여, 구현 완료 시점에 배포 테스트를 진행하였습니다.
AWS를 이용하여 데이터베이스는 AWS RDS를 통하여 배포하였고, 애플리케이션 서버는 AWS EC2를 통하여 배포하였습니다.
당시 http://skhucms.kr로 접근 가능하였으며, 이러한 노력으로 인해 교내 IT&미디어 콘텐츠 경진대회에서 은상과 인기상이라는 값진 결과를 얻을 수 있었습니다.

Spring을 이용하여 진행한 첫 프로젝트에, 복학 이후 바로 접하게 된 프로젝트였기에 초반에 많은 어려움을 겪었으나
팀원들과 서로 도와가며 진행하고 격려와 응원이 오가는 화목한 분위기 속에 진행하여 많은 도움이 되었던 것 같습니다.
