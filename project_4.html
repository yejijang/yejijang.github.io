SKHU_CMS

Spring Web 프로젝트
2020.03 ~ 2020.06 (4달간)

⋆⁺₊⋆주제

성공회대학교 동아리 관리 시스템 SKHU Club Management System - 동아리의 모집에서부터 홍보, 회의록, 회계 등 업무를 관리할 수 있는 시스템

⋆⁺₊⋆개요

SKHU_CMS는 Spring Boot를 이용하여 구현한 웹 프로젝트로, 교내의 모든 동아리 업무 관리에 편리하게 사용할 수 있습니다.
동아리 연합회는 전체 동아리의 정보를 확인하고 관리 운영할 수 있으며,
동아리 관리자는 해당 동아리 관리 및 동아리원 관리 운영할 수 있습니다.
또한 동아리원은 소속 동아리의 중요 정보를 확인하고 전체 동아리의 정보를 확인할 수 있습니다.

⋆⁺₊⋆담당업무

DB설계, 동아리 기능 Back-End, Front-End

⋆⁺₊⋆스킬

개발 환경: Windows, JDK1.8, STS, MySQL Workbench, Git
기술 키워드: Spring Boot, MySQL, MyBatis, Maven, AWS, DNS, http, Bootstrap, Gabia, tomcat, AWS EC2, AWS Route53..
