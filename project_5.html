Yumyum

Web 프로젝트 - 쌍용교육센터
2021.07.13 ~ 2021.08.06 (24일간)

⋆⁺₊⋆주제

배달 주문 웹 프로그램 - 사용자의 위치 기반으로 가게의 정보를 출력하고 주문할 수 있는 시스템

⋆⁺₊⋆개요

Yumyum은 JSP/Servlet를 이용하여 구현한 웹 프로젝트로, 배달 주문에 편리하게 사용할 수 있습니다.
사용자는 현재 위치 및 설정한 위치 기반으로 가게의 정보를 확인할 수 있으며 주문할 수 있고,
자신의 소비 내역으로 소비 패턴을 분석하여 취향을 쉽게 파악할 수 있게 구현하였습니다.

⋆⁺₊⋆담당업무

헤더 공동작업, 회원가입, 개인 정보 수정(회원 탈퇴), 주문내역 목록 출력 및 페이징, 주문내역 상세 보기, 주문내역 삭제,
가게 메뉴 선택(장바구니 추가), 장바구니 목록 출력, 장바구니 내역 삭제, 주문

⋆⁺₊⋆스킬

개발 환경: Windows, JDK1.8, Eclipse, eXERD, SQL Developer, Git
기술 키워드: Java, Oracle, JSP, Servlet, Ajax, API, JDBC, HTML5, CSS3, JavaScript, HighCharts API, 카카오 주소 API, Google geolocation API, 카카오 지도 API, JavaMail API..

⋆⁺₊⋆최종 후기

재학 중 교내 동아리 관리 시스템 프로젝트를 진행하며, 개발 시 사용자의 편의성 향상에 집중하여 우수한 평가를 받은 경험이 있습니다. 이때 교내에는 다수의 동아리와 소모임이 존재하였으나 모든 관리 기능을 수기 또는 SNS 등을 이용해 공유, 기록하는 모습을 보며 데이터의 관리가 어렵고 보존하는 데 있어 많이 불편하다는 사실을 알게 되었습니다. 각 동아리의 모집에서부터 홍보, 회의록, 회계, 회원 관리, 활동 일정 공유까지 한 페이지 내에서 관리할 수 있다면 동아리원 뿐만 아니라 총 관리자인 동아리 연합회 측에도 편리함을 줄 수 있다고 판단하여 프로젝트를 진행하게 되었습니다. MySQL로 DB를 설계하였고 Bootstrap, Spring Boot, MyBatis 등의 개발 환경을 이용하였습니다. 또한 비동기적 움직임을 위해 jQuery를 이용한 ajax, 팀원과 협업 및 버전 관리에 Git, GitHub을 사용하였습니다. 프로젝트 최종 목표인 교내 실제 구동되는 페이지로 발전시키기 위하여 실제 도메인을 구매해 동연을 비롯한 학과생들에게 배포 테스트를 진행하였고, 이러한 결과로 교내 IT 경진대회에서 은상과 인기상을 수상할 수 있었습니다. Spring을 이용하여 페이지 구상부터 구현까지 한 프로젝트도 처음이었지만, 필요성을 생각해 요구를 분석하고 사용자의 편의성을 위해 진행한 프로젝트도 처음이었기에 개발자의 역량을 키우게 된 가장 큰 경험이었다고 생각합니다. 이러한 경험을 살려 사용자의 편의성과 효율성을 보다 향상시킬 수 있는 개발자로 성장하고 싶습니다.
